# Code of Conduct

## Our commitment

We want Vowena Docs to be welcoming, respectful, and useful for contributors of all backgrounds and skill levels.

## Expected behavior

- Be respectful and generous in feedback.
- Focus on clarity, accuracy, and helping others succeed.
- Explain reasoning when suggesting structural or editorial changes.
- Treat newcomers with patience.

## Unacceptable behavior

- Harassment, discrimination, or hate speech
- Personal attacks or intimidation
- Doxxing or sharing private information without consent
- Repeatedly derailing conversations in bad faith

## Scope

This code of conduct applies across issues, pull requests, code review, discussions, and other project spaces.

## Reporting

Report conduct concerns privately through GitHub when possible. If private GitHub contact is unavailable, email `security@vowena.xyz` with the subject prefix `[conduct]`.

## Enforcement

Maintainers may edit or remove content and restrict participation when needed to keep the project safe and constructive.
