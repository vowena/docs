# Support

## Use this repository when

- A page is incorrect, missing, or hard to follow
- Navigation is broken or confusing
- A guide needs clearer examples

## Before opening an issue

- Check the repository `README.md` for local docs setup.
- Check the repository `CONTRIBUTING.md` for authoring and validation guidance.
- Check `SECURITY.md` if the issue is security-sensitive.

## Questions and routing

If you are not sure which Vowena repository owns a problem, open an issue here with enough context for maintainers to route it or move it to the right repo.
